How It Works

  • Browse the movie catalogue on the home page.
  • Click any film to see its details and read user reviews.
  • Use the Write a Review page to submit your own review.
  • All fields are validated before your review is accepted.

Tech Stack

  • Flask — Python web framework
  • Jinja2 — HTML templating (template inheritance, loops, conditionals)
  • Python — server-side data and form validation logic
  • CSS — custom stylesheet, no external libraries
