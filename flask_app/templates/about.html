Why I Built This

I built CineReview because I wanted a simple, personal way to keep track of the movies I have previously watched and forgotten about. It is frustrating to start watching a movie that my husband and I have already watched, but one or both of us have forgotten until 10-15 mins into a movie. This is not funny if it happens more than once when you sit down to enjoy something! This is especially true when you have to wait for the other person (in my case my husband) to agree you have already seen the movie (which could be 20-25 mins in).

What is CineReview?

CineReview connects to The Movie Database (TMDB) to give you access to hundreds of thousands of films — new releases, classics, and everything in between. Create an account to track what you've watched, write reviews, and get personalised suggestions based on your taste.

Who Made It?

Hi, I'm Niamh! I'm not a software developer, though for the past few months I've been learning web development and I'm starting to feel like one given the amount of debugging I've been doing. I have a passion for learning in general but also bringing ideas to life and improving other people's lives as a result. I built CineReview to solve a problem I have, but also to share my love of movies and help other people keep track of what they've watched and find new things to watch.


This project was built as part of the University College Dublin Computer Science curriculum. It is a demonstration of my skills in Python, Flask, SQLAlchemy, and web development best practices.

The source code is available on GitHub.

How to Use It

  • Browse Now Playing, Popular, Top Rated or Upcoming on the home page.
  • Use the search bar to find any film by title.
  • Open a film and click Mark as Watched to add it to your list.
  • Click Suggest a Movie for personalised recommendations.
  • Write and read reviews on every film page.

Tech Stack

  • Flask — Python web framework
  • Jinja2 — HTML templating with inheritance, loops, conditionals
  • SQLAlchemy — database ORM (SQLite locally, PostgreSQL in production)
  • Flask-Login — user authentication and sessions
  • TMDB API — live movie data, posters, and cast information
  • Render — cloud hosting platform (Web Service + PostgreSQL database)
  • Gunicorn — production web server
  • GitHub — source control and continuous deployment
